Gestión de comandas

Permite seleccionar un restaurante, elegir un plato, aplicar descuentos, registrar órdenes y gestionarlas mediante una tabla interactiva. También incluye funciones para imprimir y reiniciar las comandas.

Corral restaurante

La página es un clon de la página oficial de El Corral, desarrollado con el framework Bootstrap. Permite a los usuarios navegar, ver los productos, personalizarlos y agregarlos al carrito. El uso del acordeón facilita la organización de las categorías del menú, mientras que la barra de navegación permite una navegación rápida entre las distintas secciones del sitio.

App Music

Es una aplicación web responsiva para streaming de música, utilizando Bootstrap para el diseño y la integración de la API de Deezer para acceder a una vasta biblioteca musical.
